# housing-price-prediction(Machine Learning)

## Overview
This project builds a machine learning model to predict housing prices based on housing features.  
The goal is to explore data preprocessing, regression modeling, and model evaluation using Python.

## Dataset
The project uses the Boston Housing dataset, which includes features such as:

- Crime rate
- Average number of rooms
- Property tax rate
- Accessibility to highways
- Pupil-teacher ratio

These features are used to predict the median value of homes.

## Methods
The following steps were implemented:

1. Data loading and preprocessing
2. Feature exploration and visualization
3. Train/test split
4. Regression modeling
5. Model evaluation

## Tools & Libraries

Python  
NumPy  
Pandas  
Matplotlib  
Scikit-learn  
Jupyter Notebook  

## Results

The regression model was trained to predict housing prices based on multiple housing features.

Model performance was evaluated using common regression metrics such as:

- Mean Squared Error (MSE)
- R-squared score

The model demonstrates the ability to capture relationships between housing features and housing prices.
